# Telecom Network Digital Twin 🌐

Un jumeau numérique interactif permettant de concevoir, surveiller et simuler un réseau de télécommunications (Routeurs, Switchs, Antennes 5G, Clients) en temps réel.

## Stack Technique
- **Frontend :** React, TypeScript, React Flow, Vite
- **Backend :** FastAPI, Python, WebSockets
- **Base de données :** PostgreSQL
- **Déploiement :** Docker, Docker Compose

## Démarrage rapide
1. Cloner le dépôt.
2. Lancer `docker-compose up --build` à la racine.
3. Frontend accessible sur `http://localhost:3000`
4. Backend API accessible sur `http://localhost:8000/docs`
